# TasteRank — Design Specification

> **Version**: 1.0  
> **Date**: 2025-02-09  
> **Design System**: Tailwind 4.1 + shadcn/ui  
> **Target**: 모바일 우선 (375px ~ 430px), 데스크톱 지원  

---

## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **모바일 우선** | 모든 UI는 375px 기준 설계, 데스크톱은 max-w-lg 중앙 정렬 |
| **사진 중심** | 음식 사진이 화면의 주인공, 큰 카드/갤러리 레이아웃 |
| **원핸드 조작** | 하단 네비게이션, 중요 버튼은 엄지 영역(화면 하단 40%) |
| **최소 로그인 장벽** | 구경은 자유롭게, 수정 시에만 로그인 프롬프트 |
| **재미 요소** | 이상형 월드컵 UI, 랭킹 애니메이션, 트로피 이모지 |

---

## 3. 타이포그래피

```css
@theme {
    --font-sans: "Pretendard Variable", "Pretendard", -apple-system, sans-serif;
    --font-mono: "JetBrains Mono", monospace;
}
```

| 용도 | 크기 | 굵기 | 용례 |
|------|------|------|------|
| 페이지 타이틀 | text-2xl (24px) | font-bold | 컬렉션명, 랭킹 타이틀 |
| 섹션 헤딩 | text-lg (18px) | font-semibold | 카드 타이틀, 섹션명 |
| 본문 | text-base (16px) | font-normal | 설명, 리뷰 텍스트 |
| 캡션 | text-sm (14px) | font-normal | 날짜, 태그, 보조 정보 |
| 점수 | text-3xl (30px) | font-bold | 평균 점수, 순위 번호 |

## 4. 페이지별 디자인

### 4.1 랜딩 / 공개 컬렉션 목록 (`/`, `/trips`)

**비로그인 접근 가능**

```
┌──────────────────────────────┐
│  🍽️ TasteRank               │  ← 헤더 (로고 + 로그인 버튼)
├──────────────────────────────┤
│                              │
│  여행의 맛을 기록하세요        │  ← 히어로 텍스트 (첫 방문 시)
│  [시작하기]                   │     CTA → 로그인 or 컬렉션 생성
│                              │
├──────────────────────────────┤
│  📋 공개 컬렉션               │  ← 섹션 헤딩
│                              │
│  ┌────────────────────────┐  │
│  │ 🇯🇵 [커버 사진]         │  │  ← TripCard (세로형)
│  │  일본 여행 2025         │  │     - 커버 이미지 (aspect-video)
│  │  12개 음식 · 4명 참여    │  │     - 여행명
│  │  ⭐ 최고점 9.2          │  │     - 항목수, 멤버수, 최고점
│  └────────────────────────┘  │
│                              │
│  ┌────────────────────────┐  │
│  │  [커버 사진]            │  │  ← 다음 컬렉션
│  │  방콕 맛집 탐방          │  │
│  │  8개 음식 · 2명 참여     │  │
│  └────────────────────────┘  │
│                              │
├──────────────────────────────┤
│  [홈]   [내 여행]   [프로필]  │  ← 하단 네비게이션
└──────────────────────────────┘
```

**구성 요소:**
- 헤더: 로고 왼쪽, 로그인/프로필 버튼 오른쪽
- 히어로 섹션: 첫 방문 또는 컬렉션 없을 때만 표시
- 컬렉션 카드 리스트: 세로 스크롤, 카드 간격 gap-4
- 각 카드: 커버 이미지(aspect-video) + 정보 오버레이
- 하단 네비게이션: 3탭 (홈/내 여행/프로필), 로그인 시에만 "내 여행" 활성

### 4.2 컬렉션 상세 (`/trips/[tripId]`)

**비로그인 접근 가능**

```
┌──────────────────────────────┐
│  ← 뒤로    일본 여행 2025     │  ← 헤더 (뒤로가기 + 컬렉션명)
├──────────────────────────────┤
│  [커버 사진 - 풀 너비]        │  ← 커버 이미지 (aspect-[21/9])
│                              │
│  일본 여행 2025               │  ← 타이틀
│  2025.03.15 - 2025.03.22     │     기간
│  👥 Jeff, 아내, 친구          │     멤버 아바타 목록
│                              │
│  [음식 목록]  [랭킹]          │  ← 탭 네비게이션
├──────────────────────────────┤
│                              │
│  ── 음식 목록 탭 ──           │
│                              │
│  🏷️ [라멘] [스시] [디저트]    │  ← 태그 필터 (가로 스크롤 칩)
│                              │
│  ┌──────────┬──────────┐     │  ← 2열 그리드 (음식 카드)
│  │ [사진]   │ [사진]   │     │
│  │ 이치란   │ 스시다이  │     │     EntryCard
│  │ ⭐ 9.2  │ ⭐ 8.8   │     │     - 정사각 사진 (aspect-square)
│  │ 라멘     │ 스시     │     │     - 음식명, 평균 점수, 태그
│  └──────────┴──────────┘     │
│  ┌──────────┬──────────┐     │
│  │ [사진]   │ [사진]   │     │
│  │ ...      │ ...      │     │
│  └──────────┴──────────┘     │
│                              │
│  ＋ 음식 추가                 │  ← FAB (로그인+editor만 표시)
│                              │     비로그인 시 → LoginPrompt
├──────────────────────────────┤
│  [홈]   [내 여행]   [프로필]  │
└──────────────────────────────┘
```

**구성 요소:**
- 커버 이미지: 파노라마 비율, 그라데이션 오버레이로 텍스트 가독성
- 멤버 목록: 원형 아바타 가로 나열 (최대 5명, 이후 +N)
- 탭: "음식 목록" / "랭킹" 두 개
- 태그 필터: 가로 스크롤 가능한 칩 UI, "전체" 기본 선택
- 음식 그리드: 2열, 각 카드는 정사각 사진 + 간략 정보
- FAB (Floating Action Button): 우하단, 로그인+editor 권한 시만 표시

- 4위 이하: 리스트 형태 컴팩트 카드
- AI 랭킹 섹션: 별도 영역, 분석 코멘트 + 점수 breakdown
- 이상형 월드컵 CTA: 하단 고정 버튼 (editor 이상만)

- 위치: 기본 Geolocation, 실패 시 수동 입력 폴백
- 하단 고정 저장 버튼

## 5. 공통 컴포넌트

### 5.1 하단 네비게이션 (BottomNav)

```
┌──────────┬──────────┬──────────┐
│   🏠     │   📋     │   👤     │
│   홈     │  내 여행  │  프로필   │
└──────────┴──────────┴──────────┘
```

- 3탭 고정
- 비로그인 시: "홈" 활성, "내 여행"·"프로필" 탭 시 로그인 프롬프트
- 현재 탭 primary 색상 강조
- Safe area padding (노치/홈바 대응)

### 5.2 TripCard

- 커버 이미지: `aspect-video`, rounded-xl, 하단 그라데이션
- 정보 오버레이: 여행명, 기간, 멤버 수, 음식 수
- 탭 → 컬렉션 상세로 이동

### 5.3 EntryCard

- 음식 사진: `aspect-square`, rounded-lg
- 하단: 음식명, 평균 점수 (별 아이콘), 태그 1~2개
- 2열 그리드 배치

### 5.4 RatingSlider

- 1~10 범위, 0.5 단위
- 큰 터치 영역 (h-12)
- 현재 값 숫자로 큰 폰트 표시
- 색상: 1~3 빨강, 4~6 노랑, 7~10 초록 그라데이션

### 5.5 TagChip

- 선택 가능 칩: 선택 시 primary fill, 미선택 시 outline
- AI 추천 칩: 🤖 아이콘 접두사
- 가로 스크롤 컨테이너에서 사용

## 6. 애니메이션 & 인터랙션

| 요소 | 애니메이션 |

| FAB | scroll-down 시 축소, tap 시 회전 |

## 7. 반응형 브레이크포인트

| 화면 | 너비 | 레이아웃 |
|------|------|---------|
| 모바일 (기본) | < 640px | 1열, 하단 네비게이션, 풀 너비 카드 |
| 태블릿 | 640px ~ 1024px | 2열 그리드, 하단 네비게이션 유지 |
| 데스크톱 | > 1024px | max-w-lg 중앙, 사이드바 네비게이션 (선택) |

모바일 최적화가 최우선이므로 데스크톱은 모바일 레이아웃을 중앙에 배치하는 정도로 충분.

## 8. 접근성

- 터치 타겟: 최소 44x44px
- 색상 대비: WCAG AA (4.5:1 이상)
- 이미지 alt 텍스트: 음식명 + 식당명
- 키보드 네비게이션: 탭/엔터 지원
- 스크린 리더: 순위/점수 aria-label
